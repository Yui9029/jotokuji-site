# GitHub公開手順

## 1. ZIPをダウンロードして解凍する

ZIPファイルをダウンロードし、パソコン上で解凍します。

## 2. GitHubのリポジトリを開く

`Yui9029/jotokuji-site` を開きます。

## 3. 既存ファイルを更新する

`Add file` → `Upload files` を押します。

## 4. アップロードするもの

解凍したフォルダの中にある、以下を**そのまま全部**アップロードしてください。

- index.html
- styles.css
- script.js
- favicon.svg
- .nojekyll
- README.md
- GitHub公開手順.md
- images フォルダ

## 5. Commit changes

画面下の `Commit changes` を押します。

## 6. 公開サイトを確認する

数分待ってから GitHub Pages のURLを開きます。

反映されない場合は、ブラウザでスーパーリロードしてください。

- Windows: `Ctrl + F5`
- Mac: `Cmd + Shift + R`
- iPhone Safari: 再読込、またはタブを閉じて開き直し
