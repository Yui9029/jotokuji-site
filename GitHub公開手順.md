# GitHub公開手順

1. このフォルダの**中身だけ**をアップロードしてください。
2. リポジトリ `Yui9029/jotokuji-site` を開きます。
3. `Add file` → `Upload files` を押します。
4. `index.html`、`styles.css`、`script.js`、`404.html`、`.nojekyll`、`favicon.svg`、`images` フォルダなどをアップロードします。
5. `Commit changes` を押します。
6. `Settings` → `Pages` で `Deploy from a branch` / `main` / `/(root)` になっていることを確認します。
7. 数分後に公開URLを開きます。
