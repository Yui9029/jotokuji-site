# 浄徳寺 個別型永代供養墓 サイト

GitHub Pages 用の静的サイト一式です。

## ファイル構成

- `index.html` … 本文と構成
- `styles.css` … デザイン
- `script.js` … スマホメニュー
- `favicon.svg` … ファビコン
- `404.html` … 404ページ
- `images/` … 掲載画像

## 公開方法

GitHub リポジトリのルート直下に、この一式を置いてください。  
`images` フォルダも忘れずにアップロードしてください。

GitHub Pages の設定は以下です。

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/ (root)`
